Positioning Examples

Relative Positioning

In relative positioning, an element is positioned relative to its normal position in the document flow. It still occupies space in the document flow, but its position can be adjusted using the top, right, bottom, and left properties.

Float Positioning

Floating an element removes it from the normal document flow and allows it to be positioned to the left or right of its containing element. Other elements flow around it, similar to text wrapping around an image.

Absolute Positioning

In absolute positioning, an element is positioned relative to its nearest positioned ancestor. If no positioned ancestor is found, it is positioned relative to the initial containing block (usually the viewport). The element is taken out of the normal document flow, so it does not affect the positioning of other elements.
